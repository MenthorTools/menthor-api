# Menthor API

### Command Line Terminal

At C:\menthor-api> type:

* **Build Project**
  * gradle wrapper 
  * gradlew build
* **Generate FatJar**
  * gradle fatJar
* **Start Service** 
  * gradle bootRun
* **Access via Browser**
  * http://localhost:8080

### IntelliJ IDE

* **Opening**: IntelliJ -> Open -> /menthor-api -> OK
* **Start Service**: BootStarter.groovy -> Right Click -> Run
